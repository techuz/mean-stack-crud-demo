MEAN stack basic CRUD Application

This is a basic CRUD operation application. It is really helpfull who is in the initial stage of MEAN stack development. After running the application and looking into the code, developer will have the basic idea of Node.js, Express.js, MongoDb and Angular 4.

Prerequisites :

1) MongoDB : Install it from https://www.mongodb.org/

2) NodeJS : Install it from https:/nodejs.org/

3) AngularCLI : Install it from https://cli.angular.io/

Installation :

1) Clone this repository using git clone

2) Install NodeJs modules: Go in the root folder from terminal and run sudo npm install

3) Install Angular modules: Go in the "/meandemo/public" folder from terminal and run sudo npm install

Running Application :

1) Go in the root folder from terminal and run "nodemon index.js OR node index.js"

2) Open your browser and run http://localhost:3001
